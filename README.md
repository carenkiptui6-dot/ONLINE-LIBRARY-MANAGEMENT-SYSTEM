                                    ONLINE LIBRARY MANAGEMENT SYSTEM
  This project implements the Online Library Management System using Object Oriented Progamming Principles in C++. It allows libaies to mange users who access,borrow ad return borrowed books to the libray effortlessly.
                                 Object Oriented Progamming Design Pinciples
In this poject i implemented OOP design principles in C++ to the poject and they include:
Encapsulation-the data mambers such as book Id, author name and title are private and are accessed using public methods.
Abstraction-the classes represent real world entities(the user, book and library)
Composition- the Online Library Management system is composed of user, books, and library.
                        Features implemented
I implemented features in the system such as;
adding and removing books
seaching for books by various criteria
managing user accounts with borowing and returning capabilities.
